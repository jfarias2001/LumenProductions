# PRD-017 — BOARD V2 (funil de criação com IA) + Copy Rápida + Prompts editáveis

## Contexto / Motivação (relato do usuário)

> "Implemente um novo processo de criação com IA: sugere ideias para selecionar → sugere títulos → sugere foco (dor, oferta, etc.) → produz copy. Isso numa nova aba Kanban chamada **BOARD V2**. Crie também uma nova aba **teste** para produzir copy rápida com prompt. E os prompts precisam ser visíveis e não hardcoded: deixe os **prompts de ouro visíveis e editáveis**, e permita **prompts personalizados na hora de gerar** copy — uma opção 'prompt personalizado' ou 'prompt de ouro' antes de criar."

Hoje a criação é card a card pelo pipeline; os prompts (Regra de Ouro + guias de voz/estrutura/hooks) estão no código (`AppSetting.goldenRulePrompt` só via seed; guias como constantes no `shared`), sem UI para ver/editar nem para injetar prompts próprios na geração.

## Objetivo

1. **Funil de criação com IA** (guiado, passo a passo): **Ideias → Título → Foco → Copy**, cada passo com sugestões da IA para o usuário escolher; ao final **cria um card** já com título + foco + copy.
2. **BOARD V2** — nova aba Kanban **independente** (cards e colunas próprios) onde o funil roda e os cards produzidos ficam.
3. **Copy Rápida (aba "Teste")** — gerar copy na hora a partir de um prompt livre.
4. **Prompts editáveis (não hardcoded)** — nova aba **Prompts** para ver/editar a **Regra de Ouro** e os **3 guias** (voz da marca, estrutura criativa, hooks) + **CRUD de prompts personalizados**. Na geração (funil e copy rápida) o usuário pode **somar um prompt personalizado** à Regra de Ouro.

## Decisões (travadas com o usuário)

- **BOARD V2 = board separado/independente** — modelo de card próprio (`V2Card`) e colunas próprias; não mistura com o pipeline de 9 etapas.
- **Fim do funil = cria um card** (V2) com a copy pronta.
- **Prompt personalizado SOMA à Regra de Ouro** (a Regra de Ouro entra sempre; o personalizado é instrução extra por cima).
- **Editáveis: Regra de Ouro + os 3 guias** (voz, estrutura, hooks), além dos prompts personalizados.

## Escopo

### Dentro
- `V2Card` + colunas próprias (lean, sem gates de qualidade): **Rascunho → Copy Pronta → Aprovado → Publicado** (+ Arquivado). Funil cria em *Copy Pronta*. Kanban com drag-and-drop simples (muda a coluna).
- 4 chamadas de IA do funil (ideias, títulos, foco, copy) + 1 de copy rápida — todas passam pela Regra de Ouro + guias (do banco) e aceitam um prompt personalizado somado.
- Aba **Prompts**: editar Regra de Ouro + 3 guias (persistidos no `AppSetting`, com fallback para as constantes) + CRUD de `CustomPrompt`.
- Aba **Copy Rápida**: prompt livre → copy + CTAs, com opção de salvar como card V2.
- Seletor de prompt reutilizável ("+ prompt personalizado (opcional)": escolher um salvo ou digitar na hora) no funil e na copy rápida.

### Fora
- Gates/validação/retention no BOARD V2 (é um board leve de copy).
- Migrar os cards do pipeline atual para o V2 (coexistem).
- Versionamento/histórico de prompts.
- Trocar prompt por etapa no pipeline atual (a escolha de prompt personalizado é só no funil V2 e na copy rápida).

## Critérios de aceite

1. Aba **BOARD V2** no header; abre um Kanban próprio com as colunas do V2.
2. Botão **"✦ Criar com IA"** abre o funil: a IA sugere ideias → escolho uma → sugere títulos → escolho → sugere focos → escolho → gera a copy → **Criar card** grava um `V2Card` em *Copy Pronta*.
3. Cards do V2 podem ser arrastados entre colunas e editados (título/foco/copy/CTAs) manualmente.
4. Aba **Copy Rápida**: escrevo um prompt, escolho (opcional) um prompt personalizado, gero copy + CTAs, copio ou salvo como card V2.
5. Aba **Prompts**: vejo e edito a Regra de Ouro e os 3 guias (salvar reflete nas próximas gerações); crio/edito/excluo prompts personalizados.
6. Em qualquer geração, marcar um prompt personalizado faz a IA seguir a Regra de Ouro **e** as instruções extras.
7. Pipeline atual, gates (`PipelineService`) e cards existentes permanecem inalterados.
