# PRD-007 — Conteúdo nativo de Instagram + Auto-produção do calendário

## Contexto
Todo conteúdo da Lumen é gravado/produzido **para o Instagram** (Reels, posts de imagem única, carrosséis). Dois problemas surgiram do uso real:

1. **Estático virou "vários slides" quando era uma imagem só.** Ao gerar a direção criativa de um conteúdo **estático**, a IA devolveu uma sequência de cenas/slides — mas a peça era **uma única imagem**. O sistema tratava todo `ESTATICO` como carrossel multi-slide, sem distinguir **imagem única** de **carrossel**.

2. **O calendário parava no "enviar para o pipeline".** Cada item do calendário virava um card vazio em *Ideias Brutas*, e todo o conteúdo (ideia → ângulos → roteiro → direção → copy) ainda precisava ser gerado fase a fase, manualmente. O usuário quer que, **a partir do calendário, a IA já produza tudo** e deixe a peça **pronta para gravar/produzir**.

## Objetivo
1. Tornar a geração **nativa de Instagram** e corrigir o estático: distinguir **Imagem única** (default) de **Carrossel**; a IA decide por peça (e no calendário), com override manual.
2. Adicionar **auto-produção em lote**: um botão no calendário gera, para todos os itens ainda sem card, o pacote criativo completo e **avança cada card o máximo que os gates permitirem**, deixando-o pronto para produção.

## Decisões (travadas com o usuário)
- **Imagem única vs. carrossel:** a IA decide por peça; **default = imagem única**; só vira carrossel quando faz sentido. Usuário pode trocar manualmente.
- **Escopo do "produzir tudo":** gera **todos os entregáveis** (ideia estruturada → ângulos/hooks → roteiro → direção criativa → copy/legenda) **e avança o card** o máximo que os gates permitirem (para antes de qualquer gate que exija confirmação humana — ex.: validação).
- **Disparo:** **somente em lote** — botão "✦ Produzir tudo com IA" no calendário inteiro (processa todos os itens sem card).

## Critérios de aceite
1. Card e item de calendário têm `staticFormat` (IMAGEM_UNICA | CARROSSEL); default imagem única para estático.
2. Direção criativa de **imagem única** descreve **uma só composição** (sem slides/carrossel); carrossel mantém o slide a slide.
3. Toda geração de IA assume **Instagram** explicitamente (Reels para vídeo; feed 4:5/1:1 para imagem; carrossel até 10 cards).
4. Botão "✦ Produzir tudo com IA" no calendário: para cada item **sem card**, cria o card, gera o pacote criativo completo e avança o card até onde os gates permitem; itens com card são pulados (idempotente). Falha em um item não derruba os demais.
5. Os **gates e a revisão humana** do `PipelineService` permanecem **inalterados** — a validação gerada entra como sugestão (sem `reviewedById`), então o auto-avanço para em *Ideias Validadas*.
6. Sem `OPENAI_API_KEY` → 503 com fallback claro; chaves só no backend.
